# mvn

``` bash
  nvm -v # 查看nvm版本信息：
  nvm install 10.22.0 # 安装多版本 node/npm（以v10.22.0为例）：
  nvm use 10.22.0 # 使用特定Node版本（以v10.22.0为例）：
  nvm ls available # 查看远程服务器上的可用Node版本：
  nvm ls # 查看本机的可用Node版本
```
